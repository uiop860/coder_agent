# coder_agent

This is a [Ratatui] app generated by the [Hello World template].

[Ratatui]: https://ratatui.rs
[Hello World Template]: https://github.com/ratatui/templates/tree/main/hello-world

## Usage

When running the TUI you can type a prompt and press Enter to send it.
If the model issues a tool call you will see a `🔧 Tool call:` message in
System output.  Press **Ctrl+T** to toggle display of the tool-call arguments
(they are hidden by default), **Ctrl+R** to show/hide the reasoning
that the model emits, and **Ctrl+D** to hide or show system/debug messages.
